# 银行客户流失分析报告

## 一、项目概述

### 1.1 项目背景

在银行存量客户经营中，客户流失不仅会造成客户规模下降，还可能带来存款余额、产品收入和客户生命周期价值的损失。相比持续获取新客户，提前识别潜在流失客户并实施分层维护，通常具有更明确的经营价值。

本项目基于银行客户基础属性、账户资产、产品持有及活跃状态等数据，分析不同客群的流失表现，识别流失率较高的细分群体，为客户预警、客户触达及留存策略提供数据支持。

### 1.2 分析目标

本次分析主要回答以下问题：

1. 银行当前整体客户流失情况如何？
2. 哪些地区和性别客群的流失风险较高？
3. 客户活跃状态、资产余额和产品持有数量与流失有何关系？
4. 银行应该优先关注哪些细分客群？
5. 后续应如何衡量客户留存措施的实际效果？

------

## 二、数据说明

### 2.1 数据概况

本次分析使用银行客户流失预测数据，共包含10,000条客户记录。数据涵盖客户基础属性、账户情况、产品持有情况和是否流失等信息。

**表1 主要分析字段**

| 字段类别 | 主要字段                 | 业务含义                 |
| -------- | ------------------------ | ------------------------ |
| 客户属性 | Geography、Gender、Age   | 客户地区、性别和年龄     |
| 客户关系 | Tenure                   | 客户与银行保持关系的年限 |
| 账户资产 | Balance、EstimatedSalary | 账户余额及估算收入       |
| 产品情况 | NumOfProducts、HasCrCard | 产品持有数量及信用卡情况 |
| 行为情况 | IsActiveMember           | 客户是否处于活跃状态     |
| 结果变量 | Exited                   | 客户是否已经流失         |

### 2.2 核心指标口径

```
客户流失率 = 流失客户数 ÷ 客户总数 × 100%
```

流失率用于衡量某一客群中已经流失客户的比例。

需要注意，本报告主要进行描述性和关联性分析。某一客群流失率较高，只能说明该特征与流失存在关联，不能直接证明该特征是导致客户流失的原因。

------

## 三、执行摘要

数据集共包含10,000名客户，其中2,037名客户已经流失，整体流失率为20.4%。

分析得到以下主要发现：

1. 德国客户流失率为32.4%，较整体流失率高12.0个百分点，是三个地区中风险最高的客群。
2. 女性客户流失率为25.1%，高于男性客户的16.5%，相差8.6个百分点。
3. 非活跃客户流失率为26.9%，较活跃客户的14.3%高12.6个百分点。
4. 余额在0～5万元的客户流失率为34.7%，在资产分层中最高，但该层客户规模较小，制定策略时应同时考虑客户数量和资产价值。
5. 持有3个或4个产品的客户流失率很高，但样本量分别仅为266人和60人，需要进一步核查产品组合及业务口径。

综合来看，德国客户、女性客户及非活跃客户是优先关注对象。银行可进一步建立客户流失预警名单，并通过分层触达和对照实验评估留存措施。

------

## 四、整体流失情况

**表2 客户流失总体情况**

| 指标         | 数值   |
| ------------ | ------ |
| 客户总数     | 10,000 |
| 流失客户数   | 2,037  |
| 未流失客户数 | 7,963  |
| 整体流失率   | 20.4%  |

整体来看，每100名客户中约有20名客户流失，说明客户留存具有一定改善空间。但总体指标无法说明风险集中在哪些客群，因此需要继续从地区、性别、活跃状态、资产和产品等维度进行拆分。

------

## 五、重点维度分析

### 5.1 地区分析

德国客户流失率为32.4%，明显高于法国的16.2%和西班牙的16.7%。德国客户数量为2,509人，流失人数为814人，其流失人数甚至略高于客户规模约为其两倍的法国。

这一结果说明德国客户不仅流失率较高，而且流失规模也较大，应作为地区维度的重点分析对象。

### 5.2 性别分析

女性客户流失率为25.1%，男性客户流失率为16.5%，两者相差8.6个百分点。

但不能仅据此认定“女性更容易流失”。还需要结合地区、年龄、资产余额和活跃状态进行交叉分析，以判断性别差异是否受到其他客户特征影响。

### 5.3 活跃状态分析

**表3 客户活跃状态与流失情况**

| 客户状态   | 客户总数（人） | 流失人数（人） | 流失率（%） |
| ---------- | -------------- | -------------- | ----------- |
| 活跃客户   | 5,151          | 735            | 14.3        |
| 非活跃客户 | 4,849          | 1,302          | 26.9        |

非活跃客户流失率比活跃客户高12.6个百分点，说明客户活跃状态与流失存在明显关联。非活跃客户可以作为流失预警体系中的重要识别条件，但仍需结合客户近期交易和产品使用数据进一步验证。

### 5.4 资产分层分析

余额在0～5万元的客户流失率最高，为34.7%；10～15万元客户流失率为25.8%，15万元以上客户流失率为23.1%。

不过，0～5万元层级客户数量较少，不能只根据流失率决定资源投入。银行还需要同时考虑客户规模、账户资产和潜在价值，避免将大量资源投入到规模较小或价值有限的客群。

### 5.5 产品持有数量分析

持有1个产品的客户流失率为27.7%，持有2个产品的客户流失率仅为7.6%。持有3个和4个产品的客户流失率分别达到82.7%和100%。

后两类客户的样本量较小，因此不能直接得出“产品越多越容易流失”的结论。应进一步核查这些客户持有的具体产品组合、产品收费情况及数据定义。

------

## 六、高风险客群识别

综合地区、性别和活跃状态，可以将以下客群作为优先观察对象：

| 优先级 | 客群特征           | 风险表现             | 建议方向               |
| ------ | ------------------ | -------------------- | ---------------------- |
| 高     | 德国、女性、非活跃 | 流失率最高的组合之一 | 优先触达并开展原因调研 |
| 高     | 德国、男性、非活跃 | 流失率明显高于整体   | 提供账户及产品使用引导 |
| 中     | 女性、非活跃       | 多维度风险特征叠加   | 建立定期预警名单       |
| 观察   | 持有3～4个产品     | 流失率高但样本较少   | 核查产品组合及数据口径 |

高风险客群应同时满足“风险较高、客户规模足够、具有挽回价值”三个条件。不能只按流失率排序。

------

## 七、业务建议

### 7.1 建立分层预警机制

将地区、活跃状态、产品数量和资产余额等特征纳入预警规则，对高风险客户进行分层管理。

### 7.2 优先激活非活跃客户

针对长期无交易或产品使用频率下降的客户，设计分层触达方案，例如客户关怀、权益提醒和产品使用引导。

### 7.3 开展德国客群专项分析

进一步补充网点、渠道、费用、投诉及产品组合数据，判断德国客户高流失率是否集中在特定产品或服务环节。

### 7.4 通过实验验证策略效果

从高风险客户中随机划分实验组和对照组。实验组接受留存措施，对照组维持原有服务，比较两组后续流失率，判断策略是否真正有效。

------

## 八、效果衡量指标

后续不应只观察“触达了多少客户”，还应重点跟踪：

| 指标                     | 作用                       |
| ------------------------ | -------------------------- |
| 高风险客户流失率         | 判断目标客群是否改善       |
| 实验组与对照组流失率差异 | 判断策略是否有效           |
| 客户激活率               | 判断非活跃客户是否恢复使用 |
| 挽回客户数               | 衡量实际留存结果           |
| 单客挽回成本             | 衡量资源投入效率           |
| 挽回资产余额             | 衡量业务价值               |

------

## 九、分析局限

1. 数据为单一时间截面的客户记录，无法观察客户行为的动态变化。
2. 数据缺少交易频率、投诉、渠道使用和产品收费等信息。
3. 当前分析主要识别关联关系，不能直接证明因果关系。
4. 部分细分客群样本量较小，流失率可能不稳定。
5. 后续需要引入时间序列数据，并通过实验或预测模型验证分析结果。

这份范例最值得你模仿的不是措辞，而是每个章节都遵循：

> **数据事实 → 对比基准 → 业务解释 → 行动建议 → 风险提示**
